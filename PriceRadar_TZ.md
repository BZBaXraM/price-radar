# PriceRadar

## Описание проекта

PriceRadar — платформа для сравнения цен на товары из популярных интернет-магазинов Азербайджана.

### Источники данных

- kontakt.az
- irshad.az
- w-t.az
- bakuelectronics.az

Система должна агрегировать товары из всех магазинов, сравнивать цены и показывать пользователю наиболее выгодное предложение.

## Основной функционал

### Парсинг товаров

Для каждого товара необходимо сохранять:

- Название
- Бренд
- Категорию
- Цену
- Изображение
- Ссылку на товар
- Магазин
- Наличие

### Сравнение цен

Для одного и того же товара отображать:

- Все доступные предложения
- Минимальную цену
- Магазин с лучшей ценой

### AI Assistant

Использовать GLM-4.7-Flash (z.ai).

Ассистент должен отвечать только на вопросы, связанные с:

- товарами
- ценами
- наличием
- сравнением предложений

### Real-time обновления

Использовать WebSockets для:

- обновления цен без перезагрузки страницы
- AI-чата

### Интернационализация

Поддержка языков:

- Azərbaycan dili
- Русский
- English

### Frontend

Требования:

- Красивое отображение карточек товаров
- Изображения товаров
- Поиск
- Фильтрация
- Адаптивный дизайн

## Технологический стек

### Backend

- FastAPI
- SQLite
- SQLAlchemy
- WebSockets
- GLM-4.7-Flash API

### Frontend

- Bun
- Next.js
- TailwindCSS
- Zustand

## MVP

- Парсинг всех 4 магазинов
- Поиск товаров
- Сравнение цен
- Карточка товара
- AI Assistant
- Поддержка 3 языков
- Real-time обновления через WebSockets
